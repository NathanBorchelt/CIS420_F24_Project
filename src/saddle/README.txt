SADDLE is part of the Cluster Design Tools, which was created by Konstantin S. Solnushkin.

Please run "python example.py" to reproduce results from the SADDLE presentation.

For more information, please see: http://ClusterDesign.org/saddle/

The bundle includes "svgwrite" library by Manfred Moitzi (https://bitbucket.org/mozman/svgwrite)
and "pyparsing.py", which is the code of the pyparsing library by Paul T. McGuire (https://pyparsing.wikispaces.com/)
