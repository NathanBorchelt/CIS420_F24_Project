Short help for making automated queries:

Note: Requests are not case-sensitive.

Request
 What it does
task=help
 This page
task=design
 Run in design mode, see additional parameters below:
  racks=12
 Total number of racks to be placed
Optional parameters (floating-point values):

Rack dimensions, in metres (must be positive; default values are specified):

w=0.6
 Rack width

d=1.2
 Rack depth

Clearances, in metres (must be non-negative; default values are specified):

c_s=1
 On the sides of rack rows

c_f=1
 In front of the first row (and behind the last row)

c_a=1
 Aisle width (space between two rows)

l_xc=6
 Maximal length of a contiguous block of racks. A special case of zero instructs the algorithm to build long, inconvenient rack rows.
